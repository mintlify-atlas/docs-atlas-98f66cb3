## Prerequisites

- [Docker](https://docs.docker.com/get-docker/) and Docker Compose installed
- Git

## Run locally

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/axelcarl/go-dashboard.git
    cd go-dashboard
    ```
  </Step>

  <Step title="Create environment files">
    The project requires two `.env` files — one for the backend at the repository root and one for the frontend.

    **Root `.env`** (copy from `.env.example`):
    ```bash
    cp .env.example .env
    ```

    Then fill in the values:
    ```bash .env
    PORT=8080
    APP_ENV=development
    DB_HOST=db
    DB_PORT=5432
    DB_DATABASE=dashboard
    DB_USERNAME=postgres
    DB_PASSWORD=postgres
    DB_SCHEMA=public
    DB_SSLMODE=disable
    FRONTEND_ORIGIN=http://localhost:5173
    ```

    **Frontend `.env`** (copy from `.env.production`):
    ```bash
    cp frontend/.env.production frontend/.env
    ```

    Update `VITE_API_URL` to point to your local backend:
    ```bash frontend/.env
    VITE_API_URL=http://localhost:8080
    ```
  </Step>

  <Step title="Start all services">
    ```bash
    docker compose up
    ```

    This starts three services:
    - **db** — PostgreSQL database on port `5432`
    - **app** — Go API with live reload (Air) on `PORT` (default `8080`)
    - **frontend** — Vite dev server on port `5173`

    The first run installs npm dependencies and downloads Go modules, so it may take a minute.
  </Step>

  <Step title="Open the dashboard">
    Navigate to [http://localhost:5173](http://localhost:5173) in your browser. You should see the payment dashboard UI.

    The Go API is available at [http://localhost:8080](http://localhost:8080). Verify it is running:

    ```bash
    curl http://localhost:8080/health
    # {"message":"App is running"}
    ```
  </Step>

  <Step title="Create your first payment">
    Use the form in the dashboard UI or send a request directly to the API:

    ```bash
    curl -X POST http://localhost:8080/payments \
      -H "Content-Type: application/json" \
      -d '{"sender":"Alice","recipient":"Bob","amount":99.50}'
    ```

    The response includes the newly created payment with its `id`, `createdAt`, and `updatedAt` timestamps.
  </Step>
</Steps>

## What's running

| Service | URL | Description |
|---|---|---|
| Frontend | http://localhost:5173 | React dashboard UI |
| Backend API | http://localhost:8080 | Go REST API |
| Health check | http://localhost:8080/health | App liveness |
| DB health | http://localhost:8080/health/db | Database connectivity |
