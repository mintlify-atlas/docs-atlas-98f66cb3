---
title: 'Get payment'
description: 'Retrieve a single payment by its numeric ID.'
api: 'GET https://go-dashboard.fly.dev/payments/{id}'
---

Returns a single payment record matching the given ID.

## Path parameters

<ParamField path="id" type="number" required>
  The numeric ID of the payment to retrieve.
</ParamField>

## Response

### 200 — success

<ResponseField name="id" type="number" required>
  Unique numeric identifier for the payment.
</ResponseField>

<ResponseField name="sender" type="string" required>
  Name or identifier of the party sending the payment.
</ResponseField>

<ResponseField name="recipient" type="string" required>
  Name or identifier of the party receiving the payment.
</ResponseField>

<ResponseField name="amount" type="number" required>
  Payment amount as a floating-point number.
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of the last update.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the payment was created.
</ResponseField>

### 400 — invalid ID

Returned when the `id` path parameter cannot be parsed as an integer.

```json
{
  "error": "Invalid identifier"
}
```

### 404 — not found

Returned when no payment with the given ID exists.

```json
{
  "error": "Payment not found"
}
```

### 500 — server error

```json
{
  "error": "Failed to fetch payment"
}
```
