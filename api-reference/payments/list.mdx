---
title: 'List payments'
description: 'Retrieve all payment records.'
api: 'GET https://go-dashboard.fly.dev/payments'
---

Returns an array of all payment records. The array is empty when no payments exist.

## Response

### 200 — success

Returns an array of payment objects.

<ResponseField name="[]" type="object[]">
  Array of payment objects.
  <Expandable title="Payment object fields" defaultOpen>
    <ResponseField name="id" type="number" required>
      Unique numeric identifier for the payment.
    </ResponseField>
    <ResponseField name="sender" type="string" required>
      Name or identifier of the party sending the payment.
    </ResponseField>
    <ResponseField name="recipient" type="string" required>
      Name or identifier of the party receiving the payment.
    </ResponseField>
    <ResponseField name="amount" type="number" required>
      Payment amount as a floating-point number.
    </ResponseField>
    <ResponseField name="updatedAt" type="string" required>
      ISO 8601 timestamp of the last update.
    </ResponseField>
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 timestamp of when the payment was created.
    </ResponseField>
  </Expandable>
</ResponseField>

### 500 — server error

```json
{
  "error": "Something went wrong when fetching payments"
}
```

<RequestExample>

```bash cURL
curl --request GET \
  --url https://go-dashboard.fly.dev/payments \
  --header 'Accept: application/json'
```
